## Build a Roman Numeral Converter
Roman numerals are based on seven symbols and can be written using various combinations to represent Arabic numerals. For example:

| **Roman numerals** |	**Arabic numerals** |
| :----------: | :---------: |
|       M      |    1000     |
|       CM     |	  900      |
|       D	     |    500      |
|       CD	   |    400      |
|       C      | 	  100      | 
|       XC     |	  90       |
|       L      | 	  50       |
|       XL	   |    40       |
|       X	     |    10       | 
|       IX     |  	9        | 
|       V      | 	  5        | 
|       IV     |    4        |
|       I      | 	  1        |
